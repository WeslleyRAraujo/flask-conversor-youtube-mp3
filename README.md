### Requisitos:

### Como iniciar o projeto?
```console
git clone https://github.com/WeslleyRAraujo/flask-conversor-youtube-mp3.git
cd flask-conversor-youtube-mp3
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt
export FLASK_APP=main
flask run
```
